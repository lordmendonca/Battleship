◆ placeShip()

bool OwnGrid::placeShip ( const Ship &  ship)

Method that accumulates a ship in the container of all the ships of the player after checking the arrangement rules.

This method also places the valid ship on the own grid.

Parameters
[in]ship: A reference of the ship that was attempted to be placed.
Returns
true when ship was successfully placed, otherwise false.

The arrangement rules are checked using private helper functions namely, shipBoundary(), shipTypes(), shipIntersection() and the isValid() function of the ship itself.

The grid information is updated only after the ship is successfully placed and using a private helper function updateGridElement().
