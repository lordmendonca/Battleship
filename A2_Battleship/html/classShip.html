◆ isValid()

bool Ship::isValid ( ) const

This query method checks if the bow and stern positions for the ship provided are valid.

Returns
true if a ship is valid, else false.

A ship is valid if its bow and stern positions are valid AND

A ship is valid if it is placed either horizontally or vertically on the grid AND

A ship is valid if it has a length between 2 and 5.

◆ length()

int Ship::length ( ) const

A query method that calculates the length of a ship.

Returns
calculated length of the ship.

A length is calculated here such that ship is either placed vertically or horizontally.

Diagonal placement of the ship will result in 0 length as it is invalid placement of the ship.
