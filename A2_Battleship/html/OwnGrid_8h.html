enum gridElement_t

gridElement_t : used to represent the grid elements on own grid.

EMPTY_FIELD : has a value of 126 which is the ASCII value for '~', used to represent non-operated grid position on the board.

SHIP : has a value of 35 which is the ASCII value for '#', used to represent an occupied grid position of the ship on the grid.

GRIDPOSITION_HIT : has a value of 79 which is the ASCII value for 'O', used to represent a hit on the board.

GRIDPOSITION_MISS : has a value of 94 which is the ASCII value for '^', used to represent a miss on the board.
