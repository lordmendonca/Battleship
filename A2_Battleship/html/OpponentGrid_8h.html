enum grid2Element_t

grid2Element_t : used to represent the grid elements on opponent grid.

EMPTYFIELD : has a value of 126 which is the ASCII value for '~', used to represent non-operated grid position on the board.

SUNKENSHIP : has a value of 35 which is the ASCII value for '#', used to represent an occupied grid position of the ship on the grid.

HIT_GRIDPOSITION : has a value of 79 which is the ASCII value for 'O', used to represent a hit on the board.

MISSED_GRIDPOSITION : has a value of 94 which is the ASCII value for '^', used to represent a miss on the board.
